# Weather_app

Small web app that shows current weather for a city using the OpenWeatherMap API.

## Features
- Search city weather (temperature, humidity, wind)
- Simple responsive UI (HTML/CSS/JS)

## Prerequisites
- A browser
- (Optional) a simple static server, e.g.:
  - Python: `python -m http.server 8000`
  - Node: `npx serve .`

## Setup
1. Copy your OpenWeatherMap API key, my api key is here maybe in future i'll remove it.
2. Open `index.html` and replace the `apiKey` value in the script with your key.

## Run
- Open `index.html` in your browser, or serve the folder and browse to `http://localhost:8000`.

## Learning outcomes
- Using REST APIs from JavaScript
- DOM manipulation and showing/hiding elements (display: none / block)
- Basic CSS styling and layout (flexbox)
